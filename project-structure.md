# Project Structure – EnviroTEST

This document describes the main structure of my React Native project.

```text
EnviroTEST/
├── android/              # Android-specific project files and Gradle config
├── ios/                  # iOS-specific project files and Xcode workspace
├── node_modules/         # Installed npm packages and dependencies
├── app.json              # App configuration (name, display name, etc.)
├── babel.config.js       # Babel configuration for transforming JS/TS
├── index.js              # Entry point that registers the root component
├── package.json          # Project scripts and dependencies
├── tsconfig.json         # TypeScript configuration
├── App.tsx               # Main React Native application component
└── README.md             # Project documentation
```
